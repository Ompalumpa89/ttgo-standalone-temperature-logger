# Upload instructions

This package contains two versions:

- ttgo-temp-logger-standard.ino
- ttgo-temp-logger-powerbank.ino

Recommended GitHub structure:

ttgo-standalone-temperature-logger/
├── README.md
├── ttgo-temp-logger-standard.ino
├── ttgo-temp-logger-powerbank.ino
├── .gitignore
├── LICENSE
└── images/
    ├── prototype-overview.jpg
    ├── display-working.jpg
    └── jack-connectors.jpg

On GitHub:
1. Open the existing repository.
2. Click Add file.
3. Click Upload files.
4. Upload:
   - README.md
   - ttgo-temp-logger-standard.ino
   - ttgo-temp-logger-powerbank.ino
   - .gitignore
5. If asked, overwrite README.md and .gitignore.
6. Commit changes.

Optional cleanup:
- You can delete the older file named ttgo-standalone-temperature-logger.ino after both new files are uploaded.
- You can also delete any ZIP files that were uploaded by mistake.
