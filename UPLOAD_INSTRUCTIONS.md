# Upload instructions

This ZIP file contains the files for the GitHub repository:

ttgo-standalone-temperature-logger

Upload these files to the root of the repository:

- README.md
- ttgo-standalone-temperature-logger.ino
- .gitignore
- images/prototype-overview.jpg
- images/display-working.jpg
- images/jack-connectors.jpg

Suggested GitHub structure:

ttgo-standalone-temperature-logger/
├── README.md
├── ttgo-standalone-temperature-logger.ino
├── .gitignore
└── images/
    ├── prototype-overview.jpg
    ├── display-working.jpg
    └── jack-connectors.jpg

On GitHub:
1. Open the repository.
2. Click Add file.
3. Click Upload files.
4. Drag all files/folders from this ZIP into GitHub.
5. Commit changes.

If GitHub asks whether to replace README.md or .gitignore, choose replace/overwrite.
